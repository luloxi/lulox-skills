Skill library

Qué hace cada skill, sin vueltas.

Esta web muestra solamente de qué se trata cada skill y cómo darle el archivo al agente para que la aprenda desde GitHub. La idea es simple: ver, elegir y cargar.

No se exponen credenciales ni secretos. Solo la descripción de cada skill y un comando listo para importar el archivo SKILL.md correcto desde el repo.

Midnight

midnight-project-building

Build Midnight Network / Compact projects end-to-end with a product-first workflow: repo scaffolding, docs lookup, contract/app architecture, testing, deployment, and iterative improvement. Use when the user wants to start or evolve a Midnight project, or wants a reusable skill for Hermes, Carly, and Sancho Panza.

ProductoMidnightReusable
Comando para que tu agente la aprenda
mkdir -p ~/.hermes/skills/midnight-project-building && curl -fsSL https://raw.githubusercontent.com/luloxi/lulox-skills/main/midnight-project-building/SKILL.md -o ~/.hermes/skills/midnight-project-building/SKILL.md
